ğŸ–¼ï¸ Tampilan Antarmuka

1. Tampilan Awal Game

Menu utama yang memberikan pilihan untuk memulai permainan atau melihat instruksi.

2. Mulai Ronde (Begin Round)

Sebelum ronde dimulai, pemain diberi waktu untuk strategi dan persiapan.

3. Beli Turret

Pemain bisa membeli turret dari berbagai jenis, sesuai kebutuhan pertahanan dan uang yang dimiliki.

4. Upgrade Turret

Turret yang sudah dipasang dapat di-upgrade untuk meningkatkan damage, jangkauan, atau kecepatan tembak.

5. Game Over

Tampilan saat permainan berakhir karena markas berhasil ditembus zombie.
